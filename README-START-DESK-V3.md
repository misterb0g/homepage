# Start Desk v3

Version ciblée au-dessus de la v2b.

## Ajouts

- Message contextuel dans la pastille supérieure : salutation, date, heure, profil actif et météo si disponible.
- Fond automatique selon le moment de la journée : matin, journée, soir, nuit.
- Commandes internes dans Spotlight : `focus`, `silex`, `perso`, `code`, `apps`, `notes`, `note appeler Patrick`, `clear notes`.
- Dock conservé sans Réglages : Focus · Silex · Perso · Code · Apps · Agenda · Notes.
- Polish mobile : dock horizontal plus stable, pastille supérieure plus lisible, suppression de l’aide dans la barre sur petits écrans.
